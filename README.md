# ASRtoLLM — 媒体文件转录与摘要工具

## 快速开始

```bash
pip install -r requirements.txt
python app.py
```

打开浏览器访问 http://127.0.0.1:8080

## 使用流程

1. 在「供应商凭证管理」中填写所需供应商的 API Key 等凭证（自动保存到浏览器本地）
2. 上传音视频文件（支持 mp3/mp4/wav/m4a/webm/ogg/flac）
3. 选择 ASR 供应商和 LLM 供应商（仅显示已配置凭证的供应商）
4. 点击「开始处理」，等待转录和摘要结果

## 说明

- 所有 API Key 仅保存在浏览器 localStorage，不会存储在服务端
- 每个供应商所需的凭证字段不同，页面会根据供应商动态显示对应的输入项
